These are 10 basic html tags

  1. < b > = This tag is used to make a text bold
    2. < b > your text < /b >
  2. < body > = This tag represents the main HTML part
    3. < body > your text < /body >
  3. < br > = This tag represents a break
    4. < br > < /br >
  4. < div > = This tag represents a division or a part of an HTML document
    5. < div > < /div >
  5. < h1 > = This tag is usually used for titles
    6. < h1 > your text here < /h1 >
  6. < i > = This tag is used to make a text italized
    7. < i > your text here < /i >
  7. < img > = This tag is for images in a document
    8. < img > < img >
  8. < ol > = This tag is used for an ordered list
    9. < ol > your list goes here < /ol >
  9. < ul > = This tag is used for an unordered list
    10. < ul > your list goes here < /ul >
  10. < span > = This tag is used to style a part of a text
    11. < span > your text goes here < span >

These are 8 basic CSS properties

    Text Properties

  1. color = Sets the color of a text
    2. h1 {color: white;}
  2. text-decoration = Adds decoration to text
    3. h1 {text-decoration: line-through;}

    List Properties

  3. list-style = Sets all the properties for a list in one declaration
    4. ol {list-style: list-style-type;}
  4. list-style-image = Specifies an image as the list-item marker
    5. ol {list-style-image: URL;}

    Border Properties

  5. border = Sets all the border properties in one declaration
    6. h2 {border: border-color;}
  6. border-style = Sets the style of the four border (right, left, top, bottom)
    7. h2 {border-style: none;}

    Font Properties

  7. font-family = Specifies the font family of a text
    8. h3 {font-family: Tahoma;}
  8. font-size = Specifies the font size of a text
    9. h3 {font-size: 200px;}
